# Media Downloader

一个简单的媒体下载工具，支持音频和视频下载。

## 功能特点

- 支持查询媒体格式
- 支持音视频下载
- 支持音视频合并
- 支持音频单独下载（转换为MP3）
- 支持视频单独下载
- 支持转换为WAV格式

## 安装要求

- Python 3.6+
- yt-dlp
- ffmpeg

## 使用方法

1. 克隆仓库
2. 运行 'pip install -r requirements.txt' 安装依赖
3. 运行 'python src/downloader.py' 启动程序
4. 根据提示输入需要下载的媒体URL
